< About Me />

I caught the coding bug!

It all started one day when playing Minecraft( Yes I love Minecraft ). I thought of a feature that would be nice to add to the game but found that no one had made such a plugin.

So I decided how hard could it be and decided to make it myself. I first saw how some were made and found it immediately interesting and something clicked when I saw all of this confusing writing and decided to learn what it all meant. As I started going through video after video and as I eventually learned how to write Java I said to myself "I want more".

So I studied HTML then CSS some CSS libraries, JavaScript and its libraries such as BootStrap, Jquery, Handlebars.js, Angular.js, Node.js etc. Then my thoughts were why stop there? So I studied to become a Full Stack Developer.

I truly love this career and I'm not looking back. I wish I had found this passion sooner...

My specialties include quickly learning skills and programming languages, problem-solving, responsive design principles, website optimizations, OOP. I am still enthusiastically grabbing onto any other programming languages, frameworks, or principles I can integrate into my work...
